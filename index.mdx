Protoface renders your agent's audio as a realtime talking avatar. It works
with speech-to-speech realtime models or a separate STT/LLM/TTS pipeline:
Protoface joins the live room as an avatar participant and publishes
synchronized audio/video.

## Start here

Use the LiveKit Agents plugin for the supported integration path today. It
starts the Protoface session, connects the avatar to your LiveKit room, and
routes agent audio to the avatar.

```python
from livekit.plugins import protoface

avatar = protoface.AvatarSession(avatar_id="av_stock_001")
await avatar.start(session, room=ctx.room)
```
